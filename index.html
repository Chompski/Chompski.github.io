Northcoders news is a discussion based web-app that allows users to sign up and contribute content, it is easy to navigate and covers a range of user based topics. Antdentifier is a personal full-stack project I created, it is an application for hobbyist ant keepers, the main goal for the application is to help users identify and care for a multitude of ant species. The application achieves this by using a process of elimination, users can input their own criteria and break down the database on the backend to find the closest match for the criteria given.

The application was created using node.js, react and runs mongoDB for its back end. The project has been made fully responsive as a big part of the application was for it to be fully mobile, as such it will run smoothly on any phone and tablet.
